Navigation

This code dynamically loads HTML content from a specified URL and updates the page's main section.

Here's how it works:

Content Fetching: Sends an HTTP request to the given URL and retrieves the HTML response.

Page Content Update:

Extracts the section of the loaded HTML and replaces the content of the element with id="app" on the current page.

URL Update:

Uses the browser's history.pushState to update the URL in the address bar without reloading the page, enabling seamless navigation.

Active Link Highlighting:

Calls a function to visually update the active link based on the current URL.

Error Handling:

Catches and logs any errors during the process, such as failed requests or invalid responses. This technique is a fundamental part of Single-page applications (SPA), enabling smooth navigation and content updates without full-page reloads.
